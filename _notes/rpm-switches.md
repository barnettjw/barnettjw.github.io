| Switch                 | Description                                             |
|:-----------------------|:--------------------------------------------------------|
| `-i $_PACKAGE`{:.bash} | install package                                         |
| `-U $_PACKAGE`{:.bash} | upgrade package                                         |
| `-v`{:.bash}           | verbose                                                 |
| `-h`{:.bash}           | print hash marks (use with `-v` for nicer progress bar) |
| `-q`{:.bash}           | query installed packages                                |
| `-a`{:.bash}           | select all packages (useful with `-q`)                  |
